How to build MS SQL Server Database

This demo shows how to create a MS SQL Server Database in MS SQL Server 7.0.
  1. Open SQL Server Enterprise Manager and open the server (in this case, MELANTHA). Open Databases node, you can see all the databases on this server.

  2. Right click on Databases and select "New Database..." to create a new database.

  3. In the Database Properties dialog, enter the database name. Here the entered name is "NewDatabase". You can change other options if you want to. It is ok if you don't change anything else.

  4. Now you can see the newDatabase database under the "Databases" node, together with other databases.

  5. Now we will create a login name for the new database. Go to "Security" node and right click on "Logins", select "New Login..."

  6. In the "General" tab of the "SQL Server Login Properties" dialog box, enter Name of the login, choose "SQL Server authentication", enter your password, and change the default databse to "NewDatabase".

  7. In the "Database Access" tab, check the checkbox on the left side of the database you just created, in this case "NewDatabase". Set the "Permit in database role" to "public" and "db_owner", then click ok.

  8. You will be prompted for the password. Enter the password you chose in step 6. Now you are all set and can close the SQL Server Enterprise Manager.
