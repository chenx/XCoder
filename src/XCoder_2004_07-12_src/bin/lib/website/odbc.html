How to Create ODBC System DSN

This demo shows how to create ODBC system DSN in windows 2000 or windows XP profession.
  1. Open "Control Panel" -> "Administrative Tools" -> "Data Sources (ODBC)", and go to tab "System DSN". Click on "Add..." button will add a new System DSN record.

  2. Choose the drive for the type database you just created. In this case, it is MS SQL server, so scroll to the end of the listbox and choose "SQL Server".

  3. In the "Create a New Data Source to SQL Server" dialog box, enter the "Name" of the DSN record, in this case let's use "newDB". Choose the "Which SQL Server do you want to connect to?" to the SQL server where we just create the newDatabase database. In this case it is "MELANTHA". Click "Next" button at the bottom of the form.

  4. In the next form, select "With SQL Server authentication using a login ID and password entered by the user.", and enter the login name and password when we create the login in the "Security" node. Click on "Next" button.

  5. This is the next page, you don't need to change anything. Click on "Next" button.

  6. This is the next page, you generally don't need to change anything. Click on "Finish" button.

  7. You will be prompt with this form, which summarized your choices. Click on "Test Data Source..." button.

  8. If the test results say "TESTS COMPLETED SUCCESSFULLY!", then you are all done. Otherwise you need to go back to previous steps to check what is wrong.
